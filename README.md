# 今日运势签 🎰

一个精美的网页版运势抽签应用。

## 功能特性 ✨

- 🎴 随机运势文案和等级（大吉、中吉、小吉、末吉、凶）
- 🎨 精美的渐变背景和毛玻璃效果卡片
- ✨ 抽签时的粒子爆炸特效
- 📱 完全响应式设计，适配各种设备
- ⚡ 快速加载，无依赖

## 使用方法 🎯

1. 进入 [应用页面](https://sunnyk-elf.github.io/my-fortune-app/)
2. 点击 "抽一签" 按钮
3. 查看随机生成的运势文案和等级
4. 可以无限次重复抽签

## 技术栈 🛠️

- **HTML5** - 页面结构
- **CSS3** - 样式和动画（包括渐变、毛玻璃效果）
- **JavaScript** - 交互逻辑和粒子效果

## 运势等级说明 📊

| 等级 | 颜色 | 含义 |
|------|------|------|
| 大吉 | 🔴 红色 | 运势最好，事事顺利 |
| 中吉 | 🟢 绿色 | 运势良好，稳中有升 |
| 小吉 | 🟡 黄色 | 运势一般，有所改善 |
| 末吉 | 🔵 蓝色 | 运势低迷，需等待 |
| 凶 | 🟣 紫色 | 运势不佳，需谨慎 |
